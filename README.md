# Daily Financial Report Agent 📊

An AI-powered agent built with **LangChain + Google Gemini** that:
- Fetches **BSE Sensex**, **Nifty 50**, and **MCX Gold** prices daily
- Collects **top 5 Indian business news** headlines
- Generates an AI-written **market commentary**
- Builds a professional **`.docx` report**
- Uploads the report to **Cloudinary**
- Sends the download URL to **WhatsApp** via **Twilio**

Runs automatically every day at **9:00 AM IST** — deployable on **Railway** or **Render**.

## ⚡ Quick Start (Local)

### 1. Install dependencies
```bash
cd aditya_work
pip install -r requirements.txt
```

### 2. Configure environment variables
```bash
cp .env.example .env
# Edit .env and fill in all your API keys
```

### 3. Set up Twilio WhatsApp Sandbox
On WhatsApp, send this message to **+1 415 523 8886**:
```
join <your-sandbox-word>
```
_(Find your sandbox word in Twilio Console → Messaging → Try it out → Send a WhatsApp message)_

### 4. Test immediately (skip the 9 AM wait)
```bash
python -m agent.main_agent --test
```

### 5. Run in scheduler mode (stays alive, triggers at 9 AM IST)
```bash
python -m agent.main_agent
```

### 6. Manually trigger a run via HTTP
```bash
curl http://localhost:8080/trigger
curl http://localhost:8080/status
```

## 🚀 Deploy to Railway

1. Push this folder to a GitHub repository
2. Go to [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Select your repo
4. Go to **Variables** tab and add all keys from `.env.example`
5. Railway will auto-detect the `Procfile` and deploy

The `/health` endpoint is used by Railway as a liveness probe.

---

## 🚀 Deploy to Render

1. Push to GitHub
2. Go to [render.com](https://render.com) → New → Web Service
3. Connect your repo — Render will detect `render.yaml` automatically
4. Add environment variables in the Render dashboard
5. Deploy

## 📄 Report Contents

Each daily `.docx` report includes:

1. **Market Overview Table** — Sensex & Nifty50 with colour-coded change columns (green/red)
2. **MCX Gold Table** — 24K and 22K prices per gram and per 10g, with COMEX reference rate
3. **AI Market Commentary** — 3–4 sentences generated by Gemini
4. **Top 5 Business News** — Sourced from NewsAPI with description and source credit
5. **Disclaimer footer**

## ⚠️ Notes

- **Twilio Sandbox**: Recipients must opt-in by sending `join <word>` to the sandbox number. For production, apply for a Twilio-approved WhatsApp Business template.
- **Market Hours**: yfinance returns the last available closing price. On weekends/holidays, it returns the last trading day's closing price.
- **Render Free Tier**: Render free tier spins down after 15 minutes of inactivity. Use Railway (always-on) or upgrade Render for reliable 9 AM scheduling.
